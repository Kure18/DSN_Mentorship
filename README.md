# Electric Vehicle increase usage prediction
Problem Statement: Predicting county-level of Electric Vehicle(EV) Adoption for infrastructure planning

Objective:Build a model to forecast the EV total or percent electric vehicles 

Why it matters:Utility companies and local governments need accurate forecast of EV growth to strategically 
deploy charging stations and upgrade the electrical grid before demand spikes.

Success Metrics:
For a model to be successful,the model needs to have at least a value of 0.6-0.9 for its metrics and from both the base line model for a sample of the data and that of the entire data it was discovered that it fell below the range of 0.6-0.9.Both linear regressor and decision tree regressor did not perform well enough.

Machine Learning Metrics:
The metrics used are the mean absolute error,mean squared error because we are dealing with a continous value.
